# Blackjack React

A simple Blackjack game built with React and TypeScript. The game supports starting a new game, hitting, standing, and determines the winner using standard Blackjack rules.

## Additional Features

None. The application implements the core Blackjack game requirements only.

## Build and Run

```bash
npm install
npm run dev
```

Open the local URL displayed in the terminal (typically `http://localhost:5173`).
